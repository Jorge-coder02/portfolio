## 📦 Instalación

1. **Clonar el repositorio**
```bash
git clone https://github.com/Jorge-coder02/portfolio.git
cd portfolio

2. **Instalar dependencias**
 ```bash
 npm install
```
o si usas Yarn
```bash
yarn install
 ```

3. **Ejecutar en modo desarrollo**
  ```bash
  npm run dev
  ```
  ## El servidor estará disponible en:
  # # http://localhost:3000
